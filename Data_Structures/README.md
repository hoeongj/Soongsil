# 📚 Data Structures (자료구조)

스택·배열·재귀 등 기본 자료구조를 직접 구현하고 응용한 과목입니다. 수식 계산기, 정렬 알고리즘, 동적 배열 클래스를 Java로 작성했습니다.

> **기술 스택:** Java · 스택(Stack) · 동적 배열 · 재귀

---

## 📂 MyCalculator — 스택 기반 수식 계산기 (`MyCalculator/`)
중위 표기 수식을 후위 표기로 변환하고 계산하는 계산기입니다. (`myCalculator.java`, 실행 화면 `screenshot1~4.png`)

- **과제 목표:** 사용자가 입력한 **중위(infix) 수식**을 **후위(postfix)** 로 변환하고 그 결과를 계산한다.
- **구현 내용:**
  - 배열 기반 **`Stack` 클래스를 직접 구현**(push/pop/peek/isEmpty).
  - **중위 → 후위 변환:** 연산자 우선순위(`operRank`)와 괄호를 처리하는 변환 알고리즘(샹팅야드 방식).
  - **후위 수식 계산:** 피연산자 스택으로 `+ - * /` 평가.
  - 괄호 불일치·잘못된 토큰 등 **예외 입력 검증**(`잘못된 입력!` 처리), 정수/실수 출력 구분, 반복 실행 메뉴.
- **핵심:** 스택의 대표 응용(수식 파싱/평가), 연산자 우선순위, 견고한 입력 처리.

## 📂 Sorting — 정렬 알고리즘 4종 시각화 (`Sorting/`)
삽입·선택·병합·퀵 정렬을 구현하고 **정렬 과정을 단계별로 출력**합니다. (`sort.java` — 보고서 `report.pdf`의 소스를 코드로 복원)

- **과제 목표:** 4가지 정렬을 직접 구현하고, 각 단계에서 배열이 어떻게 변해 가는지 출력하여 동작 원리를 보인다.
- **구현 내용:** 삽입 정렬(`System.arraycopy`로 시프트), 선택 정렬(최댓값을 뒤에서부터 채움), 병합 정렬(분할 정복 + 보조 배열 병합), 퀵 정렬(첫 원소 pivot 분할). 매 단계 배열 상태를 출력하고 32개 난수 배열로 메뉴 기반 테스트.
- **핵심:** 분할 정복(재귀), in-place(삽입/선택/퀵) vs 보조 배열(병합), 정렬 과정의 시각적 추적.

> 📝 같은 정렬 주제를 **Algorithm 과목에서는 C++ 이동 의미론**으로 다루었습니다 → [../Algorithm](../Algorithm) 와 비교해 보세요.

## 📂 DoubleArraySeq — 동적 배열 시퀀스 클래스 (`DoubleArraySeq/`)
크기가 자동으로 늘어나는 `double` 시퀀스 컨테이너 클래스입니다. (`DoubleArraySeqClass.java`, 데모 `DoubleArraySeqDemo.java`, 보고서 `report.pdf`)

- **과제 목표:** 내부 배열을 동적으로 확장하는 시퀀스 ADT를 구현한다.
- **구현 내용:** 원소 추가/삽입/삭제, 용량 자동 증가(ensure capacity), 현재 원소(cursor) 탐색, 시퀀스 연결 등 컨테이너 연산.
- **핵심:** 동적 배열(가변 용량) 설계, 용량 관리(amortized 증가), ADT 캡슐화.

## 핵심 기술
- 스택 자료구조와 응용(수식 변환·평가)
- 정렬 알고리즘(삽입/선택/병합/퀵)과 분할 정복
- 동적 배열(자동 확장) 컨테이너 설계
- ADT 캡슐화, 예외/경계 조건 처리
